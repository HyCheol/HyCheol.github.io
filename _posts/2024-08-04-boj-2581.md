## 요약
* 링크 : <https://www.acmicpc.net/problem/2581>
* 분류 : 수학, 정수론, 소수 판정
* 시간 제한 1초
* 메모리 제한 128MB

## 문제
* 자연수 M과 N이 주어질 때 M이상 N이하의 자연수 중 소수인 것을 모두 골라 이들 소수의 합과 최솟값을 찾는 프로그램을 작성하시오.
* 예를 들어 M=60, N=100인 경우 60이상 100이하의 자연수 중 소수는 61, 67, 71, 73, 79, 83, 89, 97 총 8개가 있으므로, 이들 소수의 합은 620이고, 최솟값은 61이 된다.

## 입력
* 입력의 첫째 줄에 M이, 둘째 줄에 N이 주어진다.
* M과 N은 10,000이하의 자연수이며, M은 N보다 작거나 같다.

## 출력
* M이상 N이하의 자연수 중 소수인 것을 모두 찾아 첫째 줄에 그 합을, 둘째 줄에 그 중 최솟값을 출력한다. 
* 단, M이상 N이하의 자연수 중 소수가 없을 경우는 첫째 줄에 -1을 출력한다.

## 풀이
* M부터 소수인 수를 탐색, 소수가 나온 경우 개수의 누적치 저장
* 제일 처음 나온 소수를(누적치가 0인 경우) 최솟값으로 저장
* 범위 내 모든 소수의 합을 sum_prime 변수에 누적
* sum_prime이 0인 경우 -1 출력

## 제출
* 언어 : C++
* 시간 : 148ms
* 메모리 : 2020KB
* 코드 길이 : 459B

```cpp
#include <iostream>
using namespace std;

int main() {
	int m;
	int n;
	int min_prime = 0;
	int sum_prime = 0;
	int cnt_prime = 0;

	cin >> m >> n;

	for (int i = m; i <= n; i++) {
		int cnt_factor = 0;
		for (int j = 1; j <= i; j++) {
			if (i % j == 0) cnt_factor++;
		}
		if (cnt_factor == 2) {
			if (cnt_prime == 0) min_prime = i;
			sum_prime += i;
			cnt_prime++;
		}
	}

	if (cnt_prime != 0) cout << sum_prime << endl << min_prime;
	else cout << -1;
}
```
